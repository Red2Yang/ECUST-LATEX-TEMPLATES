# ECUST-LATEX-TEMPLATES

华东理工大学，课程论文常用模板

## 简介

随着人工智能的发展，生成式AI接管了很多工作，甚至包括论文的写作。但是，人工智能无法解决格式的问题，因为传统文档软件都是基于图形化页面进行操作，无法通过生成内容直接应用，导致文档不符合规范。而tex文件就没有这种问题，因为文档通过编程的形式写作，可以被生成。不过，在一些细节上人工智能尚做得不足，该模板可以起到规范的作用。

本文所使用格式均为教务处、学校老师发布的相关格式内容。

## 如何使用

本地使用：安装`Texlive`后，会安装tex文件编辑器，打开 [src](src/) 内的文件，修改内容即可。

Overleaf：导入 [src](src/) 内tex文件即可。
