# ECUST-LATEX-TEMPLATES

华东理工大学，课程论文常用模板

## 简介

随着人工智能的发展，生成式AI接管了很多工作，甚至包括论文的写作。但是，人工智能无法解决格式的问题，因为传统文档软件都是基于图形化页面进行操作，无法直接通过生成内容直接应用，导致格式不匹配。

## 如何使用

本地使用：安装`Texlive`后，会安装tex文件编辑器，直接打开[src](src/)内的文件，修改内容即可。

Overleaf：导入[src](src/)内tex文件即可。
